
# ecdysis
snake game with evolution
## Installation
via Brackets
```
click nyo lang yung live preview
```

via XAMPP
git clone nyo lang sa loob ng <xampp-directory>/htcdocs/ecdysis
   ```
   git clone https://github.com/edgel0rd/ecdysis.git D:/xampp/htdocs/ecdysis
   ```
